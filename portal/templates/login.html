Расширение для Chrome

Вы можете использовать наше расширения для ускоренной и отложенной загрузки контента с ближайших точек доступа.

Инструкция по установке
Инструкция по установке плагина Ternex

  • Скачайте архив с плагином по ссылке ниже, а затем извлеките его содержимое.

  • Перейдите в раздел расширения, в браузере Chrome.

  • Выберите режим разработчика в правом верхнем углу.

  • В появившемся меню, выберите "Загрузить распакованное расширение" и укажите папку с расширением.
